# 【超簡単】スマホで使えるようにする方法

## 方法：Netlify Drop（ドラッグ&ドロップだけ！）

GitHubもコマンドも不要です。

### 手順（5分で完了）

#### 1. Netlifyにアクセス
https://app.netlify.com/drop にアクセス

#### 2. フォルダをドラッグ&ドロップ
- エクスプローラーで以下のフォルダを開く：
  ```
  c:\Users\k_shibuya\Example folder\pokemon-quiz\dist
  ```
- `dist`フォルダを、Netlifyのページにドラッグ&ドロップ

#### 3. 完了！
数秒後、URLが表示されます。
例：`https://random-name-12345.netlify.app`

このURLをスマホのブラウザで開けば、アプリが使えます！

---

## 補足

### アカウント登録（任意）
- ドラッグ&ドロップだけなら登録不要
- 登録すると、URLを変更したり、サイトを管理できます

### 更新する場合
1. コードを修正
2. `npm run build` を実行
3. 新しい `dist` フォルダを再度ドラッグ&ドロップ

---

## トラブルシューティング

### 「dist」フォルダが見つからない
- `npm run build` が成功しているか確認
- `pokemon-quiz` フォルダ内に `dist` フォルダがあるはず

### サイトが表示されない
- ブラウザのキャッシュをクリア
- 数分待ってから再度アクセス
